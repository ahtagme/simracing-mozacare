🚀 Setup Awal — 30 Menit

1Edit nomor WA di landing page

Buka landing_page.html pakai Notepad atau VSCode. Cari semua 62XXXXXXXXXX dan ganti dengan nomor WA kamu (format Indonesia tanpa angka 0 depan).

Contoh: HP 0812-3456-7890 → ganti jadi 6281234567890

Total ada ~7 link wa.me yang harus di-ganti. Pakai Find & Replace (Ctrl+H) biar cepat.

2Test landing page lokal

Double-click landing_page.html — akan kebuka di browser. Test:

  • Klik tombol "Order via WhatsApp" → harusnya buka WhatsApp Web/app dengan pesan ke nomor kamu
  • Klik tombol Tiktok/IG/Saweria → arahkan ke akun kamu
  • Cek di mobile: tampilannya harus tetap rapi
3Upload landing page ke internet (HOSTING GRATIS)

Pilih salah satu (paling mudah):

Option A: Netlify Drop (paling cepat — 1 menit)

  • Buka app.netlify.com/drop
  • Drag & drop file landing_page.html ke kotak besar
  • Tunggu 5 detik — dapat URL: random-name-12345.netlify.app
  • Copy URL ini
  • (Opsional) Sign up free → custom subdomain jadi ahta-service.netlify.app

Option B: Beli domain (paling pro — Rp 150K/tahun)

  • Beli domain ahta.id atau ahtaservice.com di Niagahoster/Domainesia
  • Connect ke Netlify (panduan bawaan)
  • URL kamu jadi: ahta.id
4Pasang link di bio TikTok @ahtaaa_

Buka TikTok → Edit Profile

  • Bio (max 80 char):
🛠️ Setup wheel: 75K
🎮 Install game: 50K
📦 PAKET: 250K (hemat!)
📩 Order ↓
  • Website / Link: paste URL Netlify/domain kamu
5Setup WhatsApp Business + Auto Reply

Install WhatsApp Business (Play Store / App Store, gratis):

  • Settings → Business Tools → Greeting message → aktifkan
  • Paste template #1 dari wa_templates.txt
  • Settings → Quick Replies → tambah shortcut buat template lain
6Bookmark order_tracker.html di browser

Double-click order_tracker.html → Ctrl+D (bookmark)

Setiap dapat order baru, buka tracker → Add Order → simpan.

⚠️ Data tracker disimpan di localStorage browser. Jangan clear browser data tanpa Export Backup dulu!

🎯 Pro Tips

1. Up-sell ke @mozacare shop

Setelah customer puas dengan jasa kamu, tawarin upgrade gear:

"Wheel-mu udah maksimal, tapi pedal-nya masih basic ya?
Coba upgrade ke pedal load cell, kerasa beda banget.
Cek di shop saya @mozacare ada diskon 10% buat customer:
kode 'AHTA10' pas checkout."

2. Repeat customer = goldmine

Setelah 1 bulan, follow up customer lama. Tawarin:

  • Tune-up FFB ulang (preset season baru)
  • Mod terbaru
  • Setup buat game baru rilis

20% repeat customer = revenue stable tanpa effort marketing baru.

3. Bundle dengan akun @mozacare

Beli wheel di @mozacare = setup gratis dari @ahtaaa_

Strategi: customer bayar wheel + setup sekaligus. Convert higher karena value.

4. Bikin testimoni jadi konten

Setiap customer puas → minta testi → screenshot → upload jadi video TikTok.

Format: "Customer ke-[N] — setup G29 di Jakarta. Hasilnya:" + clip before/after.

Social proof = lebih banyak customer baru.

5. Naikin harga setiap 3 bulan

Quarter 1: Rp 75K (sekarang)
Quarter 2: Rp 100K (after 30+ testimoni)
Quarter 3: Rp 150K (after build authority)
Quarter 4: Rp 200K (after exclusive partnership brand)

Customer baru bayar lebih, customer lama dapat "loyalty price" — win-win.
